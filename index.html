Missing Values

Preprocessing
Exam tip: Always motivate your method — mean/median for numeric, mode for categorical. Confirm afterwards with df.isnull().sum().

Fill with mean (numeric)

# Single column
df['column'].fillna(df['column'].mean(), inplace=True)

# All numeric columns at once
for col in df.select_dtypes(include='number').columns:
    df[col].fillna(df[col].mean(), inplace=True)
print(df.isnull().sum())

Fill with median (numeric, skewed data)

df['column'].fillna(df['column'].median(), inplace=True)

Fill with mode (categorical / text)

df['column'].fillna(df['column'].mode()[0], inplace=True)

Fill with group mean (e.g. same vehicle name)

# Augment missing values with average of the same vehicle name group
df['mileage'] = df.groupby('name')['mileage'].transform(lambda x: x.fillna(x.mean()))

# For all numeric cols using group mean
for col in ['mileage', 'engine', 'max_power']:
    df[col] = df.groupby('name')[col].transform(lambda x: x.fillna(x.mean()))
print(df.isnull().sum())

Drop rows with missing values

df.dropna(inplace=True)
print(df.isnull().sum())   # confirm

Calculate missing from other columns (e.g. chess moves)

# Derive number_of_moves from the moves string
df['number_of_moves'] = df['all_moves_in_standard_chess_notation'].apply(
    lambda x: len(str(x).split()) if pd.notna(x) else 0
)

Remove outliers using IQR (recommended method)

for col in ['Weight', 'Length', 'Width', 'Thickness']:
    Q1 = df[col].quantile(0.25)
    Q3 = df[col].quantile(0.75)
    IQR = Q3 - Q1
    lower = Q1 - 1.5 * IQR
    upper = Q3 + 1.5 * IQR
    df = df[(df[col] >= lower) & (df[col] <= upper)]

print("Shape after removing outliers:", df.shape)

Cap outliers (clip) instead of removing

for col in ['Weight', 'Length']:
    Q1 = df[col].quantile(0.25)
    Q3 = df[col].quantile(0.75)
    IQR = Q3 - Q1
    df[col] = df[col].clip(Q1 - 1.5*IQR, Q3 + 1.5*IQR)

Preprocessing
Rule of thumb used in all papers: 2 unique values → binary [0,1] | 3–10 unique values → label encode [0,1,2…] or one-hot | >10 unique values → one-hot encode

Binary encoding (2 unique values → 0/1)

from sklearn.preprocessing import LabelEncoder
le = LabelEncoder()

for col in df.select_dtypes(include='object').columns:
    if df[col].nunique() == 2:
        df[col] = le.fit_transform(df[col])
print(df.head())

Label encoding (3–10 unique values → 0,1,2…)

for col in df.select_dtypes(include='object').columns:
    if 2 < df[col].nunique() <= 10:
        df[col] = LabelEncoder().fit_transform(df[col])

# Or specific columns by name

Preprocessing
Over-sampling = create synthetic minority samples. Use when dataset is small and you can't afford to lose data. Under-sampling = remove majority samples. Use when dataset is large enough.

Learning Curves

ML

Plot learning curve for one classifier

from sklearn.model_selection import learning_curve

def plot_learning_curve(estimator, title, X, y, cv=10):
    train_sizes, train_scores, val_scores = learning_curve(
        estimator, X, y, cv=cv,
        train_sizes=np.linspace(0.1, 1.0, 10),
        scoring='accuracy', n_jobs=-1
    )
    train_mean = np.mean(train_scores, axis=1)
    val_mean   = np.mean(val_scores, axis=1)

    plt.figure(figsize=(8, 5))
    plt.plot(train_sizes, train_mean, '--', label='Training score')
    plt.plot(train_sizes, val_mean, label='Cross-validation score')
    plt.xlabel('Training Set Size')
    plt.ylabel('Accuracy Score')
    plt.title(title)
    plt.legend()
    plt.show()

# Draw for each classifier
plot_learning_curve(GaussianNB(), 'Learning Curve for the NB Classifier',  X_train, y_train)

Manual Precision, Recall, F1 (for closed-book theory)

# From ST1-2024 Q1.6:
# Positive class = at risk (12 actual), Predicted positive = 8
# Of those 8 predicted: 5 correct, 3 wrong (false positives)
# TP=5, FP=3, FN=7 (12-5), TN=?

# Precision = TP / (TP + FP)
precision = 5 / (5 + 3)   # = 0.625
print(f"Precision: {precision:.4f}")

# Recall = TP / (TP + FN)
recall = 5 / (5 + 7)      # = 0.4167
print(f"Recall:    {recall:.4f}")

# F1 = 2 * (precision * recall) / (precision + recall)
f1 = 2 * (precision * recall) / (precision + recall)
print(f"F1 Score:  {f1:.4f}")

Gini Impurity (Decision Tree Theory)

Theory
Formula: Gini(node) = 1 − Σ(pᵢ²)   |   Weighted Gini = (n_left/n_total)×Gini_left + (n_right/n_total)×Gini_right   |   Choose feature with the lowest weighted Gini.

Calculate Gini Impurity for HBP / Age < 48 example

# Data from exam: 7 records, HA = [Y,N,N,N,Y,N,Y]
# HBP = Y: rows 1,3,5,6,7  → HA: Y,N,Y,N,Y  → 3 Yes, 2 No
# HBP = N: rows 2,4         → HA: N,N         → 0 Yes, 2 No

def gini(p_yes, total):
    p1 = p_yes / total
    p0 = 1 - p1
    return 1 - (p1**2 + p0**2)

# HBP split
g_hbp_yes = gini(3, 5)      # 5 records where HBP=Yes
g_hbp_no  = gini(0, 2)      # 2 records where HBP=No
weighted_hbp = (5/7)*g_hbp_yes + (2/7)*g_hbp_no
print(f"Weighted Gini (HBP):     {weighted_hbp:.4f}")

# Age split: Age<48=Y: rows 1,2,3,4,6 → HA: Y,N,N,N,N → 1Y,4N
#            Age<48=N: rows 5,7         → HA: Y,Y         → 2Y,0N
g_age_yes = gini(1, 5)
g_age_no  = gini(2, 2)
weighted_age = (5/7)*g_age_yes + (2/7)*g_age_no
print(f"Weighted Gini (Age<48):  {weighted_age:.4f}")

# The feature with the LOWER Gini is the root
root = "HBP" if weighted_hbp < weighted_age else "Age < 48"
print(f"Root feature: {root}")

Text Preprocessing for Sentiment Analysis

NLP
Key steps to know for theory questions: Lowercasing → Remove punctuation/special chars → Tokenisation → Stop word removal → Stemming/Lemmatisation → Vectorisation (TF-IDF or BoW)

Full NLP preprocessing pipeline

import re
import nltk
from nltk.corpus import stopwords
from nltk.stem import PorterStemmer, WordNetLemmatizer
from nltk.tokenize import word_tokenize
nltk.download('stopwords')
nltk.download('punkt')
nltk.download('wordnet')

stop_words = set(stopwords.words('english'))
stemmer    = PorterStemmer()
lemmatizer = WordNetLemmatizer()

def preprocess_text(text):
    text = text.lower()                              # 1. Lowercase
    text = re.sub(r'[^a-z\s]', '', text)            # 2. Remove punctuation
    tokens = word_tokenize(text)                     # 3. Tokenise
    tokens = [t for t in tokens if t not in stop_words]  # 4. Remove stopwords
    tokens = [lemmatizer.lemmatize(t) for t in tokens]  # 5. Lemmatise
    return ' '.join(tokens)

df['clean_text'] = df['review'].apply(preprocess_text)

# 6. Vectorise (TF-IDF)
from sklearn.feature_extraction.text import TfidfVectorizer
tfidf = TfidfVectorizer()
X     = tfidf.fit_transform(df['clean_text'])
